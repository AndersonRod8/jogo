
## Arquivos sons utilizados

### colisao.wav

Efeito sonoro reproduzido quando a bola colide com uma das barras ou com os limites superior e inferior da tela.

### ponto.wav

Efeito sonoro reproduzido quando um dos jogadores marca ponto.

## Formato dos arquivos

Os efeitos sonoros são armazenados no formato WAV (.wav), adequado para sons curtos e de rápida reprodução.

## Observações

Os arquivos devem permanecer na pasta:

assets/sons/

para que sejam carregados corretamente pelo sistema de áudio do jogo.
