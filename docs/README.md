# Documentação

Esta pasta concentra documentos de planejamento e apoio ao projeto.

## Arquivos

- `proposta.MD`: proposta inicial do jogo para preenchimento do grupo.

## Decisões de Desenvolvimento

- O projeto foi desenvolvido em Python utilizando a biblioteca Pygame.
- O código foi organizado em módulos para facilitar a manutenção.
- Os efeitos sonoros foram armazenados na pasta `assets/sons`.
- O histórico das partidas é salvo em arquivo texto.

## Funcionalidades Implementadas

- Controle de dois jogadores.
- Sistema de pontuação.
- Movimentação da bola.
- Detecção de colisões.
- Efeitos sonoros.
- Registro do histórico das partidas.
